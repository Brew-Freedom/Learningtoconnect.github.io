# How to Contribute

We welcome everyone.

## You can contribute by:
- Writing a story about a real experience
- Sharing communication tips
- Giving advice on friendships

## Steps:
1. Fork the repository
2. Create a new file in `_posts/`
3. Use this format:

---
layout: post
title: "Your Title"
category: stories
---

Your content here.

4. Submit a Pull Request

## Guidelines
- Be respectful
- No hate speech
- Keep it real, not perfect
# Code of Conduct

This is a safe space.

## Be kind
## Be respectful
## No harassment or discrimination

We’re here to support—not judge.
