# Learning to Connect

A community project for people who struggle with communication and maintaining friendships.

## Why this exists
Many of us find it hard to:
- start conversations
- maintain friendships
- express ourselves clearly

This project is a shared space to learn together.

## 🤝 How to contribute
You can:
- share a personal story
- give advice that helped you
- write guides for specific situations

No expertise needed—just honesty.

See CONTRIBUTING.md for details.
